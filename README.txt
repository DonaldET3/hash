This repository contains file hashing programs that are under development, except for the ones contained in the "old" directory.

There are currently ten file hashing programs under development here each one implements a different hash function, though some of them are blended variations of each other.

* BLAKE
* CubeHash
* Keccak
* MD6
* Simon
* SimonBLAKE
* SimonCube
* SimonSkein
* Skein
