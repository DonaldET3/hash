OUmanifest extensions for Skein hash functions
version 1

function name for Skein: skein

specification fields
The "rounds" field specifies the number of rounds used to process the data. (default: 72)
The "size" field specifies the number of bits in the hash output. (default: 512)
The "wordcount" field specifies the number of words used in the hash state. (power of 2, at least 4, default: 8)
