OUmanifest extensions for SimonCube hash functions
version 1

function name for SimonCube: simoncube

specification fields
The "init" field specifies the number of rounds used to initialize the function.
The "rounds" field specifies the number of rounds used to process the data.
The "block" field specifies the number of bytes per data block (up to 128).
The "final" field specifies the number of rounds used to finish the hash.
The "size" field specifies the number of bits in the hash output. (a multiple of 8, at least 8, not more than 512)
